# Timer Pro

**Timer Pro** is a high-performance HIIT and Circuit Training timer built with pure Vanilla JavaScript. It features a distraction-free **Quick Timer** for simple countdowns and a powerful **Circuit Mode** for complex interval workouts.

Key capabilities include a drag-and-drop workout editor, custom iOS-style time pickers, active theme states (Red/Blue/Green) for visual feedback, and a completely dependency-free architecture. Designed for athletes who need precision, focus, and zero latency during training sessions.
